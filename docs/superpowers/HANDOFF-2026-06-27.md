## O que é

Fonte única de verdade das taxas de financiamento imobiliário (MCMV/SBPE). Um script Node/TS roda
numa GitHub Action semanal, raspa gov.br + BCB e — **só quando muda** — commita o JSON versionado,
servido via CDN. Sem banco, sem servidor: o git é o "banco" (cada commit = versão auditável).
Porte da lógica que vivia no `engaja-amiz` (Edge Functions Deno + Supabase) para um serviço neutro.
Custo R$ 0.

## Decisões tomadas nesta sessão

| Tema | Decisão |
|------|---------|
| Arquitetura | GitHub Actions (cron) + JSON commitado + jsDelivr. Sem Supabase/DB/Cloudflare. |
| Indexadores BCB | Atualizam mensalmente (entram no gatilho de mudança) **com guarda anti-zero**: falha do BCB nunca zera bons valores. |
| `rulesStale` | Recalculado **no cliente** por idade (>21 dias). O serviço serve o JSON cru. Helper `withStaleness` no README. |
| Stack | TypeScript + tsx + Vitest. Sem etapa de build. |
| Action | Cron `0 11 * * 1` (seg 08h BRT) + `workflow_dispatch`; `npm test` como guarda; commit condicional; purge jsDelivr best-effort. |
| Contrato | `RatesPayload` **idêntico** ao engaja — consumidores não mudam. |
| Detecção de mudança | `contentHash` = sha256 do `parsed` (faixas/classe-média); `changed` se faixas OU indexers válidos mudam. Sem commit de ruído por timestamp. |
| Licença | MIT (dado público do gov.br). |

## Riscos conhecidos (registrados, não acionáveis agora)

- Parser por regex sobre HTML achatado: se o gov.br reordenar colunas mantendo valores plausíveis
  (0–20%), `isPlausible` passaria com valores trocados. Mitigado por layout estável + auditoria via
  commit. Mudar a estratégia exigiria sair do escopo fixado na spec.
- jsDelivr `@main` tem cache de borda ~12h; o purge na Action acelera. Para travar versão, usar `@v0.1.0`.
